# Project 2 market research

Deep research compiled 2026-07-09, scanning 16 live remote AI engineering postings (Reedsy, n8n,
Viktor, Starbridge, Huzzle, Duvo, ShipBob, JUPUS, Action1, Cohere, SoSafe, Zapier, Bullpen Talent,
Scale Army, ANNA Autism Care) to determine the best-fit second portfolio project.

**Conclusion**: the recurring pattern across postings is "ship production-grade AI workflows,
agents, and evaluation loops inside real business systems" — not another chatbot/demo. Recommended
project: an **AgentOps Control Room** — import contacts, run configurable AI workflows across
voice/text/email, log every event/transcript/tool-call, score outcomes with rule + LLM-judge
evals, dashboard the results, export via webhook.

Key skill-frequency findings across the 16 postings:
- TypeScript/full-stack JS: 9/16
- Agents/workflows/automation: 7/16
- Python: 6/16
- APIs/integrations/webhooks: 6/16
- React: 6/16
- Evals/quality/regression: 5/16
- LLMs/prompting: 5/16
- Cloud/deployment/infra: 5/16
- Retrieval/RAG/vectors: 2-3/16
- Voice AI: 3/16 explicit

Full posting-by-posting table, company profiles, DB schema draft, API route draft, and shipping
timeline are preserved in the original research output (not re-copied here in full — see git
history / conversation log if needed). This file exists so the "why" behind this project's scope
